Copyright, Creative Common (CC) Licenses, Fair Use

Basic Copyright Principles:

  • Ownership of original works protects creators' rights.
  • Exclusive rights to reproduce, distribute, and create derivatives.
  • Copyright duration is typically life plus 50-70 years.
  • Infringement may result in legal penalties.
  • Exceptions like fair use or Creative Commons apply.

4 Types of Creative Commons (CC) Licenses:

  • CC BY: Free use with attribution.
  • CC BY-SA: Attribution and same license for derivatives.
  • CC BY-ND: No derivatives allowed.
  • CC BY-NC: Non-commercial use only.

Fair Use in New Zealand:

Fair use permits limited use of copyrighted works for purposes like research, commentary, criticism, and parody.

  • Creative Commons Licenses
  • Copyright Council of New Zealand

Privacy & Privacy Policy

What is a Privacy Policy?

A legal document explaining how personal data is collected, used, and protected by an organization.

13 Principles of Privacy (NZ):

  • Collection must be for lawful purposes.
  • Data should be collected directly from the individual.
  • Individuals must be informed about the data collection.
  • Data collection must be fair and non-intrusive.
  • Data security must be ensured.
  • Individuals have the right to access their data.
  • Individuals can correct inaccurate data.
  • Data must be accurate and up to date.
  • Data retention should only last as long as necessary.
  • Data use should be limited to the purposes for which it was collected.
  • Data should not be disclosed without consent.
  • Unique identifiers should be used minimally.
  • Cross-border data transfers must follow protection laws.

Sources:

  • NZ Privacy Act 2020 Principles
  • NZ Office of the Privacy Commissioner

Web Publishing

5 Factors to Consider When Choosing Web Hosting:

  • Reliability and uptime: Ensure 99.9% or higher uptime.
  • Speed and performance: Fast server speeds are essential.
  • Customer support: 24/7 availability across multiple channels.
  • Security: Features like SSL, backups, and malware protection.
  • Scalability: Choose a host that can grow with your needs.

Web Performance & Maintenance

5 Ways to Reduce Page Load Time:

  • Optimize and compress images to reduce file size.
  • Minify CSS and JavaScript code.
  • Leverage browser caching for faster repeat visits.
  • Use a Content Delivery Network (CDN) to distribute content.
  • Reduce HTTP requests by combining files.

5 Website Maintenance Tasks:

  • Perform regular backups to prevent data loss.
  • Keep software, plugins, and scripts updated to avoid security issues.
  • Regularly update content to stay relevant and improve SEO.
  • Check for broken links and fix them to enhance user experience.
  • Monitor website performance using tools like Google Analytics.

Sources:

  • Google Developers - Site Speed
  • WebFX - Ways to Speed Up Your Website

Web Security

5 Common Cyber Attacks & Preventive Measures:

  • Phishing: Educate users and use strong spam filters.
  • SQL Injection: Use prepared statements and input validation.
  • Cross-Site Scripting (XSS): Validate and sanitize user inputs.
  • Denial of Service (DoS): Implement firewalls and load balancers.
  • Man-in-the-Middle Attacks: Use SSL/TLS encryption for secure communication.

Sources:

  • CISA - Cybersecurity
  • Cloudflare - Web Security
